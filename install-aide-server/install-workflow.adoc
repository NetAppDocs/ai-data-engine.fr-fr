= Flux d'installation et de configuration pour AI Data Engine sur des serveurs tiers
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Pour installer et configurer votre AI Data Engine software sur des serveurs tiers, vous installez et câblez les composants matériels, mettez le système sous tension et configurez votre cluster ONTAP.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Un"]link:aide-mde-server-hardware-requirements.html["Examinez les exigences d'installation du matériel"]
[role="quick-margin-para"]
Assurez-vous de disposer d'un système de stockage AFX 1K déjà installé, puis consultez les exigences matérielles pour l'installation de serveurs tiers ou de nœuds de calcul de données fournis par NetApp. Pour plus d'informations sur l'installation du système de stockage AFX 1K, reportez-vous à link:https://docs.netapp.com/us-en/ontap-afx/install-setup/install-setup-workflow.html["Documentation d'installation du système de stockage AFX 1K"^].

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Deux"]link:deploy-hardware.html["Installez les serveurs tiers"]
[role="quick-margin-para"]
Les serveurs tiers doivent être installés conformément aux instructions fournies par le fabricant pour votre modèle de serveur spécifique.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Trois"]link:cable-hardware.html["Câblez vos serveurs tiers"]
[role="quick-margin-para"]
Pour câbler le matériel, connectez d'abord les serveurs tiers à votre réseau de données, puis connectez les serveurs tiers aux commutateurs du cluster.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Quatre"]link:power-on-hardware.html["Mettez sous tension vos serveurs tiers"]
[role="quick-margin-para"]
Après avoir installé le rack et câblé votre matériel, vous devez mettre sous tension vos serveurs tiers et vos nœuds de contrôle pour le système de stockage AFX s'ils ne sont pas déjà allumés.
